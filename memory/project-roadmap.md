## Fase siguiente — pendiente (post mayo 2026)

El CRM está operativo al 100%. Estas son mejoras planificadas para una próxima fase:

- Reportes exportables más completos (PDF/Excel con más datos y gráficos)
- Notificaciones push
- Integración WhatsApp API

**Why:** El usuario quiere mantener estas ideas para cuando decida continuar el desarrollo.
**How to apply:** Cuando el usuario retome el desarrollo, partir de estas ideas como punto de entrada para la siguiente fase.
